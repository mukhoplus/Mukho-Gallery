# Mukho Gallery

묵호 갤러리~~(이제 갤러리 기능이 없는)~~

## 기능

- 로그인
- 회원가입
  - 이메일 인증
  - 카카오 지도 주소 검색
- 아이디 찾기
- 비밀번호 찾기
  - 이메일 인증

## 기술 스택

### FE

- TypeScript
- Next.js
- Tailwind

### BE & DevOps

- Kotlin
- Spring Boot
- JPA
- MySQL
- (Oracle Cloud Infrastructure, Nginx, Docker, Jenkins, Certbot)
